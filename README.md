# Sentrum

## Getting started

Clone this repo

```bash
yarn
yarn start
```

### macos

#### pre-requisites to compile into apps

```shell
brew install wine64
brew cask install wine-stable
brew cask install xquartz wine-stable
brew install mono
brew install fakeroot
brew install dpkg
brew install rpm
brew install rpmbuild
```
